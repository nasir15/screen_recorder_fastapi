Important setup notes
Windows: system audio usually works best with a loopback capture device such as virtual-audio-capturer.
Linux: system audio usually uses PulseAudio / PipeWire monitor sources. Try default first, or a monitor source from pactl list short sources.
macOS: true system-audio capture usually needs a loopback driver like BlackHole.
